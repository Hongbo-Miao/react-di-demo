# React DI

Demo for implementing Angular's DI in React. The original of this project could be found here [https://github.com/shibe97/React-MineSweeper](https://github.com/shibe97/React-MineSweeper).

![MineSweeper](https://shibe97.github.io/assets/images/ss/ss_minesweeper_01.png)

In `node_modules` you can find the forked version of `react-dom` which uses [injection-js](https://github.com/mgechev/injection-js) (the extraction of the Angular's DI).

## Setup

```
$ git clone https://github.com/mgechev/react-di-demo.git
```

## Build

```
$ npm run build
```

## Watch and auto build

```
$ npm run watch
```
