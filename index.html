React-MineSweeper

How To Play

  1. 1. Select a difficulty level. If you're new to MineSweeper, start with easy.
  2. 2. The number in the upper-right corner denotes the number of mines on the board.
  3. 3. The number on the left is the timer.
  4. 4. Find where the bomb is. And open a cell you think there is not the bomb.
  5. 5. If you open all the cell except for the bombs, you will win.
